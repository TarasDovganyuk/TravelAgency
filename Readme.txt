Welcome to TravelAgency website project!

1) to start you need to change to your parameters in resources/hibernate.properties

jdbc.username
jdbc.password

2) create database MySQL "travelagency" and run script in resources/database.sql

3) by default user with login "manager" and password "11111111" with 
administrative privilegies is created.

4) Use "manager" to create hotels, rooms 

5) create users to book rooms in your hotels

6) by default rooms can be econom, standart, superior and luxury types

7) if you want ot add two or more rooms same type - just create them separately
